# How does the script work?
You clone the repository to your Kali system and grant permissions with `chmod +x` in the `install_claude_kali.sh` script.
After that, you run it using `./install_claude.sh`

# Clone repo
`git clone https://github.com/kaikys0usa/install_claude_kali.sh`

# Update kali
Use `sudo apt update && sudo apt upgrade -y` to update the operating system and its dependencies

# Permission & Run
Navigate to the location of the script and grant permission using `chmod +x install_claude_kali.sh`, then execute it with `./install_claude.sh`.
